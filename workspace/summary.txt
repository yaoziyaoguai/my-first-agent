# 项目说明

## 项目简介
这是一个基于 Claude API 的智能 Agent 项目，能够实现多轮对话、工具调用和文件操作功能。

## 主要依赖

### API 客户端
- **anthropic (0.89.0)**: Claude API 的官方 Python 客户端
- **httpx (0.28.1)**: 现代化的 HTTP 客户端
- **httpcore (1.0.9)**: HTTP 核心库

### 数据验证与配置
- **pydantic (2.12.5)**: 数据验证和序列化
- **python-dotenv (1.2.2)**: 环境变量管理

### 异步支持
- **anyio (4.13.0)**: 异步 I/O 抽象层
- **sniffio (1.3.1)**: 异步框架检测

## 核心功能 (05_agent_with_files.py)

该文件实现了具有文件操作能力的智能 Agent：

1. **多轮对话**: 支持与 Claude 进行持续交互
2. **工具调用**: 
   - `read_file`: 读取文件内容
   - `write_file`: 写入/覆盖文件内容
   - `calculate`: 数学计算
3. **文件操作**: 能够自动创建目录、备份原文件
4. **错误处理**: 完善的异常处理和日志输出

## 项目结构
- 依赖管理: requirements.txt
- 主程序: 05_agent_with_files.py
- 输出目录: workspace/

## 运行环境
- Python 3.x
- 需要 ANTHROPIC_API_KEY 环境变量
